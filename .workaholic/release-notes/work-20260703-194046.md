## Key Changes

- **Terminal `CALL` is now an effect:** `drive.copy` performs a server-side copy (owner md5-verified
  round-trip) and `mail.send` sends a real message; both preview as a `CALL` and apply on `--commit`.
  `drive.copy` takes a destination **folder path** (`parent_path`), resolved at apply time.
- **SQLite DBMS management:** `CREATE TABLE` / `REMOVE TABLE` (parser sugar, no new keywords) create
  and drop tables â€” drop behind the irreversible gate â€” and reading `/sql/<conn>` lists one
  `{name, kind}` row per relation (`SHOW TABLES`). `DESCRIBE` reflects a just-created table.
- **Gmail drafts write parity:** a draft INSERT must name its columns (`values (to, subject, body) â€¦`);
  the ambiguous positional form is rejected at **plan time** so preview and commit agree, and the
  `/mail/drafts` collection no longer advertises a set-wide `REMOVE` the applier can't honor.
- **Gmail read fidelity:** `/mail |> select name` shows label **display names** (not `Label_5`), and a
  message or draft read now lists its real **attachments** (filename/mime/size).
- **Gmail date search works:** `where date < '2024-01-01'` and `where date BETWEEN 'a' AND 'b'` are
  coerced and pushed to Gmail `after:`/`before:` instead of being silently ignored.
