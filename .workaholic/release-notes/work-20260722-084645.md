# Per-row decode over collected sets and retire the markdown driver etc

## Summary

File collections become declared sets over any blob source: DECODE runs per-row over collected sets with path provenance, markdown documents/links are codec relations registered as declared views with a wired read-by-path mount, and the compiled /markdown driver is retired after byte-for-byte row-equivalence. Cookbook collection recipes are now execution-checked.
