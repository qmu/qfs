Settled the reference divergences in ADR 0010: **do not** adopt the engine's `multiColumn` (its stated `mapMsg`/SSR tension does not exist — the real blocker is that `Scene` cannot express a level's body, so calling it renders "Not found" everywhere), and fixed the root page shipping **zero** `main` landmarks while restoring the reference's kind-driven rule for shallower columns with compile-time exhaustiveness.
